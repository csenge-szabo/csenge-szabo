# Welcome! 👋

**AI Engineer** at [onefold.ai](https://onefold.ai) · Based in Vienna, Austria

---

I design and ship production AI systems for enterprise customers — end to end. On any given week that means multi-agent orchestration with LangGraph, hybrid RAG pipelines (dense + sparse retrieval over pgvector), FastAPI services streaming over WebSockets, Next.js frontends, and the Azure infrastructure-as-code underneath it all. Alongside custom customer solutions, I contribute to internal product design and development. I'm passionate about responsible AI, multi-agentic systems, and using technology for meaningful societal impact.
